    Class TextInput

    Represents a text input component builder.

    Example

    const textInput = new TextInput().setLabel("Enter text");
    textInput.setStyle(TextInputStyle.Paragraph);
    textInput.setPlaceholder("Type here");
    const json = textInput.toJSON();
