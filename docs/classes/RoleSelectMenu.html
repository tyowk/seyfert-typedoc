    Class RoleSelectMenu

    Represents a Select Menu for selecting roles.

    Example

    const roleSelectMenu = new RoleSelectMenu();
    roleSelectMenu.setCustomId("role-select");
    roleSelectMenu.addDefaultRoles("123456789", "987654321");
